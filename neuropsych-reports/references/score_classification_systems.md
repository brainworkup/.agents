# Neuropsychological Score Classification Systems

Reference for normative classification frameworks used in neuropsychological report writing.

---

## Primary Classification Systems

Used for: WAIS-V, WISC-VI, WMS-IV, WIAT-4, and most SS-based measures

| Standard Score | Percentile Range | Classification |
|---------------|-----------------|----------------|
| ≥ 130 | ≥ 98th | Very Superior |
| 120–129 | 91st–97th | Superior |
| 110–119 | 75th–90th | High Average |
| 90–109 | 25th–74th | Average |
| 80–89 | 9th–24th | Low Average |
| 70–79 | 2nd–8th | Borderline |
| ≤ 69 | ≤ 1st | Extremely Low |

Used for: TMT, Grooved Pegboard, FAS, BNT (Heaton norms)

| T-Score | Classification |

| > 59 | Above Average |
| 40–59 | Average |
| 35–39 | Low Average |
| 30–34 | Mildly Impaired |
| 25–29 | Mildly-to-Moderately Impaired |
| 20–24 | Moderately Impaired |
| < 20 | Severely Impaired |

## Base Rates of Low Scores in Healthy Populations

Understanding how many low scores to expect in a comprehensive battery is essential for avoiding over-diagnosis.

| Battery Length | Expected # of Scores ≤ 5th %ile (healthy adults) |
|---------------|--------------------------------------------------|
| 5 tests | 0–1 |
| 10 tests | 0–2 |
| 15 tests | 1–3 |
| 20 tests | 1–4 |
| 25 tests | 2–5 |
| 30+ tests | 2–6 |

**Key reference:** Binder, Iverson, & Brooks (2009). Base rates of abnormal neuropsychological test performance in healthy adults.

**Clinical implication:** A single low score in a comprehensive battery may represent normal variability rather than true impairment. Look for convergent evidence (multiple low scores within the same domain, consistency with clinical history and daily functioning).

## Reliable Change Index (RCI) Thresholds

For serial evaluations, use RCI to determine whether score changes represent true change beyond measurement error and practice effects.

**General guidelines (vary by test and retest interval):**
- Change of ≥ 1.0 SD (15 SS points) typically exceeds RCI at 90% confidence
- Consult test-specific RCI tables (available in most test manuals)
- Account for practice effects (especially for measures like TMT, Stroop, CVLT)
- Regression to the mean affects extreme initial scores

## Demographically Adjusted Norms

When available, consider using demographically adjusted norms (correcting for age, education, sex, and/or race/ethnicity) in addition to age-only norms.

**Common demographically adjusted norm sets:**
- Heaton norms (age, education, sex, race)
- NAB (age, education, sex)
- NACC UDS3 neuropsychological battery norms
- Mayo Older Americans Normative Studies (MOANS)

**When to use:**
- When the referral question involves decline from premorbid level
- When demographic factors significantly differ from the standardization sample
- Always report which norms were used

**Caveats:**
- Demographically adjusted norms may obscure true impairment in groups with historically lower educational access
- Consider reporting both adjusted and unadjusted scores when demographic correction is substantial
- Clinical judgment should integrate normative data with individual context
