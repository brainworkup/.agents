# Validity Testing (PVT/SVT)

> Performance Validity Tests (PVTs) and Symptom Validity Tests (SVTs) assess whether test results are valid and interpretable. Essential before interpreting cognitive data, especially in forensic and disability contexts.

## Performance Validity Tests (PVTs)

Assess whether the patient is putting forth adequate effort during cognitive testing.

| Test | Format | Valid Cutoff |
|------|--------|-------------|
| Test of Memory Malingering (TOMM) | 50-item forced-choice | Trial 2 ≥ 45 |
| Word Memory Test (WMT) | Computerized forced-choice | IR, DR, CNS ≥ 82.5% |
| Medical Symptom Validity Test (MSVT) | Computerized forced-choice | IR, DR, CNS ≥ 85% |
| b Test | Paper-pencil | Errors ≤ 25 |
| Reliable Digit Span (RDS) | Embedded in Digit Span | ≥ 7 (some use ≥ 8) |
| Rey 15-Item Test (FIT) | 15-item recall | ≥ 9 (with recognition) |
| Dot Counting Test (DCT) | Time-based | Ungrouped-grouped difference, E-score |

---

## Symptom Validity Tests (SVTs)

Assess whether self-reported symptoms are consistent and credible.

| Measure | Key Validity Scales |
|---------|-------------------|
| MMPI-3 | F, Fp, Fs, FBS-r, RBS, TRIN, VRIN |
| PAI | NIM, PIM, MAL, DEF, RDF, INC, INF |
| BASC-3 | F Index, Consistency, Response Pattern |
| Conners-4 | Inconsistency, Negative Impression, Positive Impression |

---

## When to Administer

- **Always** in forensic evaluations
- **Always** in disability or litigation contexts
- **Recommended** in all comprehensive evaluations
- Embedded PVTs (RDS, CVLT recognition) should be used even in clinical settings

---

## Interpretive Principles

1. **PVT failure does NOT equal malingering** — consider all explanations
2. Invalid PVT = test results are **not interpretable** for that domain
3. Multiple PVTs provide stronger evidence than a single measure
4. Use both free-standing and embedded PVTs
5. Document validity findings prominently in the report
6. Report cutoffs used and reference sources
