# Writing Recommendations

> Providing specific, actionable, individualized recommendations tailored to the patient's cognitive profile and functional needs.

## Related Topics
- [[report-writing-guide]]
- [[test-results-interpretation]]

---

## Principle

**Avoid boilerplate. Every recommendation must be tailored** to the individual's specific cognitive profile, history, and functional context.

---

## Recommendation Categories

| Category | Examples |
|----------|----------|
| **Medical/Neurological** | Referrals (neurology, psychiatry), imaging, labs, medication review |
| **Psychological/Psychiatric** | Therapy modality, medication consultation, support groups |
| **Cognitive Rehabilitation** | Therapy, external aids (calendars, smartphones), environmental modifications |
| **Educational** | IEP/504, accommodations (extended time, reduced workload), tutoring |
| **Vocational/Occupational** | Vocational rehab, workplace accommodations, fitness for duty |
| **Safety/Functional** | Driving evaluation, capacity assessment, supervision needs |
| **Lifestyle** | Exercise, sleep hygiene, nutrition, cognitive engagement |

---

## Format

Numbered list, specific and actionable:

```
1. **Medical:** [Patient] is referred to neurology for further evaluation
   of [specific concern]. Brain MRI may help clarify [differential].

2. **Cognitive Rehabilitation:** [Patient] would benefit from cognitive
   rehabilitation therapy focused on [specific domain(s)]. External
   memory aids such as [specific examples] are recommended.

3. **Educational:** Given the above findings, [Patient] is recommended
   for a Section 504 Plan with the following accommodations:
   extended time (time-and-a-half) on tests, preferential seating
   away from distractions, and written instructions for complex tasks.
```

---

## Writing Tips

| Good | Weak |
|------|------|
| "Cognitive rehabilitation therapy focused on compensatory memory strategies" | "Consider cognitive therapy" |
| "Extended time (time-and-a-half) on tests" | "Accommodations as appropriate" |
| "Written instructions for multi-step tasks" | "Written instructions" |
| "Referral to neurology for evaluation of progressive memory decline" | "Follow up with doctor" |

---

## Key Points

- Number recommendations
- Make them specific and actionable
- Tailor to the individual's profile
- Connect to test findings
- Include timeframe when appropriate
- Categorize for clarity
