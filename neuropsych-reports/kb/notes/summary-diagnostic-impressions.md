# Summary and Diagnostic Impressions

> Synthesizing all findings into a cohesive clinical narrative with diagnostic formulation.

## Structure

1. **Opening summary** — Restate referral and demographics (1–2 sentences)
2. **Cognitive profile summary** — Strengths, weaknesses, pattern (1 paragraph)
3. **Integration with history** — Cognitive findings related to medical/psychiatric history
4. **Diagnostic formulation** — Primary and differential diagnoses with codes
5. **Functional implications** — Impact on daily living, school, work, social
6. **Validity and limitations** — Factors qualifying interpretation

---

## Diagnostic Formulation Format

```
**Diagnostic Impressions:**

1. [Primary Diagnosis] ([DSM-5-TR code], [ICD-10 code])
2. [Secondary Diagnosis] ([codes])
3. [Additional diagnoses as applicable]

**Differential diagnoses considered:**
- [Diagnosis ruled out]: [Rationale]
```

---

## Example Language

**Cognitive profile summary:**
```
Testing revealed a cognitive profile characterized by [relative strength]
and [relative weakness]. [Patient] demonstrated [domain-specific findings].
This pattern is [consistent with / suggestive of / not consistent with]
[diagnostic consideration].
```

**Functional implications:**
```
These findings suggest [Patient] may have difficulty with [specific
functional tasks]. [He/She/They] would likely benefit from [general
accommodation type].
```

---

## Key Points

- Synthesize, don't just list
- Connect findings to referral questions
- Use "suggests" or "consistent with" not "proves"
- Include both DSM-5-TR and ICD-10 codes
- State limitations clearly
- Address validity in final paragraph
