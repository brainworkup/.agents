## Normative Classification Reference

### Standard Scores (Mean = 100, SD = 15)

| Range | Classification |
|-------|---------------|
| ≥ 130 | Very Superior |
| 120–129 | Superior |
| 110–119 | High Average |
| 90–109 | Average |
| 80–89 | Low Average |
| 70–79 | Borderline |
| ≤ 69 | Extremely Low |

### T-Scores (Mean = 50, SD = 10)

| Range | Classification |
|-------|---------------|
| ≥ 70 | Very Superior |
| 60–69 | Above Average |
| 40–59 | Average |
| 30–39 | Below Average |
| 20–29 | Impaired |
| < 20 | Severely Impaired |
