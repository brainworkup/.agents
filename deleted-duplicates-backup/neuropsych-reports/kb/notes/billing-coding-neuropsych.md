## CPT Codes

| Code | Description | Typical Use |
|------|-------------|-------------|
| 96132 | Neuropsych testing evaluation, first hour | Clinician testing, first hour |
| 96133 | Neuropsych testing, each additional hour | Additional clinician testing time |
| 96136 | Test admin/scoring by psychologist, first 30 min | Psychologist-administered |
| 96137 | Test admin/scoring, additional 30 min | Additional psychologist time |
| 96138 | Test admin/scoring by technician, first 30 min | Psychometrist-administered |
| 96139 | Test admin/scoring by technician, additional 30 min | Additional technician time |
| 96146 | Automated/electronic test administration | Computerized testing |

## Record Retention

- **Adults:** Typically 7 years after last contact (varies by state)
- **Minors:** Until age of majority + state retention period
- Retain: informed consent, test protocols, raw data, report, correspondence, billing records
