# Heaton T-Score Classifications

> Neuropsychological norms classification system based on T-scores (M=50, SD=10), commonly used for tests normed via Heaton norms including TMT, Grooved Pegboard, FAS, and BNT.

## Classification Table

| T-Score | Classification |
|---------|---------------|
| > 59 | Above Average |
| 40–59 | Average |
| 35–39 | Low Average |
| 30–34 | Mildly Impaired |
| 25–29 | Mildly-to-Moderately Impaired |
| 20–24 | Moderately Impaired |
| < 20 | Severely Impaired |

---

## Score Metrics

- **Mean:** 50
- **Standard Deviation:** 10
- **Direction:** Higher = better performance

---

## Commonly Used With

| Test | Measures |
|------|----------|
| Trail Making Test (A & B) | Processing speed, cognitive flexibility |
| Grooved Pegboard | Fine motor dexterity |
| COWA / FAS | Verbal fluency |
| Boston Naming Test | Confrontation naming |
| Stroop Color and Word Test | Inhibition, processing speed |

---

## Key Differences from Wechsler System

- **Broader Average range:** T 40–59 spans ±1 SD vs. Wechsler's ±0.67 SD
- **Impairment language:** Uses "Impaired" rather than "Extremely Low"
- **Demographic adjustments:** Heaton norms correct for age, education, sex, and race
- **Clinical threshold:** Scores ≤ 34 (below 5th percentile) generally warrant clinical attention

---

## Clinical Notes

- T-score of 40 = 16th percentile (−1 SD)
- T-score of 30 = 2nd percentile (−2 SD)
- Always specify which norm set was used (Heaton vs. test-specific norms)
- Heaton norms are demographically adjusted — note this in reports
