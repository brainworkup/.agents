# Reason for Referral Section

> How to write the opening section of a neuropsychological report, establishing why the evaluation was requested and what questions it aims to answer.

## Related Topics
- [[report-writing-guide]]
- [[summary-diagnostic-impressions]]

---

## Purpose

State:
1. **Why** the evaluation was requested
2. **By whom** (referral source)
3. **What questions** the evaluation aims to answer

---

## Content Elements

| Element | Description |
|---------|-------------|
| Referral source | Name, credentials of referring provider |
| Specific questions | What the referrer wants to know |
| Patient concerns | Patient's/family's primary concerns in their words |
| Evaluation purpose | Differential diagnosis, treatment planning, accommodations, forensic, disability |
| Presenting problems | Brief description of cognitive/behavioral concerns |

---

## Writing Style

- **1–2 paragraphs**
- **Third person, past tense**
- **Professional, neutral tone**

---

## Template

```
[Patient] is a [age]-year-old [handedness], [language-dominant], [sex] who was
referred for neuropsychological evaluation by [referring provider, credentials] to
assess current cognitive functioning in the context of [presenting concern].

The evaluation was requested to [clarify diagnosis / establish baseline / guide
treatment planning / determine accommodations eligibility / assess decision-making
capacity]. [Patient/family] reported concerns regarding [chief complaints].
```

---

## Key Points

- Be specific about referral questions
- Include patient/family perspective
- Set up the rest of the report
- Do not include diagnostic conclusions
