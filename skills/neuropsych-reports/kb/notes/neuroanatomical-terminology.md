# Neuroanatomical Terminology

> Brain regions and their associated cognitive functions, commonly referenced in neuropsychological report interpretation and clinical correlation.

## Brain Regions and Functions

| Region | Functions | Relevance |
|--------|-----------|-----------|
| Hippocampus | Memory encoding | Atrophy in Alzheimer's |
| Prefrontal cortex | Executive function, planning, inhibition, working memory | TBI, frontal lobe dementias |
| Temporal lobe | Memory, language (dominant), facial recognition (non-dominant) | Alzheimer's, TBI |
| Parietal lobe | Visuospatial processing, calculation, body awareness | Neglect, vascular |
| Frontal lobe | Executive function, motor planning, personality, social behavior | TBI, bvFTD |
| Occipital lobe | Visual processing | Lewy body dementia |
| Basal ganglia | Motor control, procedural learning | Parkinson's, Huntington's |
| Cerebellum | Motor coordination, timing, some cognition | TBI, stroke |
| Corpus callosum | Interhemispheric transfer | MS, TBI, callosotomy |
| White matter | Neural connectivity | Vascular disease, MS, TBI |

---

## Clinical Terminology

| Term | Meaning |
|------|---------|
| Cortical | Pertaining to cerebral cortex (e.g., Alzheimer's = cortical dementia) |
| Subcortical | Below cortex (e.g., vascular, Parkinson's = subcortical dementia) |
| Lateralized | Affecting one hemisphere (left = language; right = visuospatial) |
| Anterior | Front; executive, motor, personality |
| Posterior | Back; sensory, visual, language comprehension |

---

## Clinical Profiles

| Profile | Pattern | Common Etiologies |
|---------|---------|-------------------|
| Cortical | Amnesia, aphasia, agnosia, apraxia | Alzheimer's, FTD |
| Subcortical | Slowing, executive dysfunction, retrieval deficits | Parkinson's, vascular, MS |
| Frontal | Disinhibition, poor planning, apathy | TBI, bvFTD |
| Temporal | Memory loss, language deficits | Alzheimer's, TLE |
