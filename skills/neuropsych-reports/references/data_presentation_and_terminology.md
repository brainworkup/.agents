# Data Presentation and Terminology for Neuropsychological Reports

Adapted from general clinical data presentation standards, tailored for neuropsychological evaluation documentation.

---

## Score Table Design Principles

### General Guidelines

- Clear, descriptive title (e.g., "Table 1. Neuropsychological Test Results: Attention and Executive Function")
- Column headers with score metric and units (e.g., "Standard Score (M=100, SD=15)")
- Left-align measure/subtest names; right-align or center numeric scores
- Footnotes for abbreviations, norms used, and accommodations
- Consistent decimal places within each table
- Consistent classification terminology throughout the report
- Number tables sequentially if using multiple tables

### Neuropsych-Specific Table Formats

**By Domain (Recommended for narrative integration):**
One table per cognitive domain, placed within the corresponding interpretive section.

**Comprehensive Appendix Table:**
All scores in a single table at the end of the report, organized by domain. Useful for quick reference by readers familiar with the measures.

**Hybrid Approach:**
Summary scores in narrative sections; full subtest breakdowns in an appendix.

### Formatting Conventions

**Bold or flag impaired scores:**
- Bold scores ≤ 5th percentile (≤ SS 75)
- Or use asterisk (*) with footnote
- Helps readers quickly identify deficit areas

**Confidence intervals:**
- Include 95% CI for composite/index scores when clinically meaningful
- Especially useful for IQ scores and when evaluating change over time

**Multiple norm sets:**
- If both age-corrected and demographically adjusted norms are reported, use separate columns or tables
- Clearly label which norms are used

### Table Examples

**Standard Score Table:**
```
Table 1. Intellectual Functioning (WAIS-V)

Index                          Standard Score   95% CI      Percentile   Classification
────────────────────────────────────────────────────────────────────────────────────────
Full Scale IQ (FSIQ)           92              87–97       30th         Average
Verbal Comprehension (VCI)     105             99–111      63rd         Average
Visual Spatial (VSI)           88              82–95       21st         Low Average
Fluid Reasoning (FRI)          95              89–101      37th         Average
Working Memory (WMI)           82              76–89       12th         Low Average
Processing Speed (PSI)         78              72–87       7th          Borderline
```

## Figure Design for Neuropsych Reports

### Cognitive Profile Graphs

**Bar charts** or **line graphs** displaying scores across domains provide a visual summary of the cognitive profile.

**Design:**
- X-axis: Cognitive domains or test indices
- Y-axis: Standard score (or T-score) scale
- Horizontal reference line at the mean (100 or 50)
- Shaded band for Average range (SS 90–109 or T 40–59)
- Color coding: green (Average+), yellow (Low Average), red (Borderline/Impaired)
- Error bars for 95% CI if applicable
- Colorblind-safe palette

**Caption example:**
```
Figure 1. Cognitive Profile Summary

Standard scores across major cognitive domains for [Patient]. The shaded band
represents the Average range (SS 90–109). Scores below the band indicate
relative weakness; scores above indicate relative strength. Error bars
represent 95% confidence intervals.
```

### Comparison Figures (Re-evaluation)

For follow-up evaluations, side-by-side bar charts or line graphs showing baseline vs. current scores:

**Design:**
- Grouped bars (baseline vs. current) for each domain
- Indicate reliable change with symbols (↑ improved, ↓ declined, = stable)
- Note practice effects in caption

### Memory Learning Curve

For CVLT-3, RAVLT, or similar list-learning tests, plot trials 1–5 as a line graph:

**Design:**
- X-axis: Trials (1–5, List B, Short Delay, Long Delay)
- Y-axis: Number of words recalled
- Compare to normative expectation if available
- Include interference trial (List B) and delay trials

## Statistical Presentation in Neuropsych Reports

### Presenting Group Comparisons (Research/Forensic)

When comparing a patient's performance to normative expectations:
- Report standard score, percentile, and classification
- Note whether score represents a statistically significant deviation from the mean (typically ≥ 1.5 or 2 SD)
- Note whether score represents a clinically meaningful change from estimated premorbid level

### Reliable Change Reporting (Re-evaluations)

For serial assessments, report:
- Baseline and current scores side by side
- Score difference
- Whether the change exceeds the RCI threshold for the specific measure
- Whether practice effects have been accounted for
- Clinical significance of the change (not just statistical significance)

**Example:**
```
Table. Comparison of Baseline (2024) and Current (2026) Testing

Domain              Baseline SS   Current SS   Difference   Exceeds RCI?
──────────────────────────────────────────────────────────────────────────
FSIQ                98            95           -3           No (WNL)
VCI                 105           102          -3           No (WNL)
WMI                 92            78           -14          Yes (Decline)*
PSI                 88            82           -6           No (WNL)
CVLT-3 LDFR (T)     45            30           -15          Yes (Decline)*

*Change exceeds the 90% confidence interval for reliable change, accounting
for practice effects (Iverson, 2001).
```

### Percentile Explanations for Non-Specialist Readers

Include a brief, plain-language explanation in the report for non-psychologist readers:

```
Percentile ranks indicate the percentage of same-age individuals in the
normative sample who scored at or below the patient's level. For example,
a score at the 25th percentile means the patient performed as well as or
better than 25% of same-age peers. Scores between the 25th and 75th
percentile are considered within the Average range.
```
